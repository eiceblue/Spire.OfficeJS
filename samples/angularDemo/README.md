# angularDemo

This demo demonstrates how to quickly integrate OfficeJS (v10.11.4) into an Angular application.

## Environment Setup
| Name   | Version      |
| ------- | --------- |
| nodejs  | `22.12.0` |
| angular | `20.2.0`  |

## OfficeJS Resource Location
The web module of the OfficeJS package is placed in the /public/spire.cloud folder for this project. In actual development, you may place these resources in any directory of your choice.

## Install dependencies
``` bash
npm install
```

## Run
``` bash
npm run start
```
