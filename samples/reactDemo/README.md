# reactDemo

This demo shows how to integrate OfficeJS (v10.11.4) into a React application.

## Environment Setup
| Name   | Version      |
| ------ | --------- |
| nodejs | `22.12.0` |
| react  | `19.1.1`  |
| vite   | `7.1.7`  |

## OfficeJS Resource Location
The web module of the OfficeJS package is located in /public/spire.cloud/ for this demo. In actual development, you may place these resources in any directory of your choice.

## Install dependencies
``` bash
npm install
```

## Run
``` bash
npm run dev
```

## Access URL
http://localhost:5174
